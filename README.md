#### 5. 如果要設計 dark mode，該如何設計？

視時間需求而定，若時間較長則會架構主題色調利用並使用 CSS(--value)、SCSS(\$value)或是 Stylus(value = )等賦值方式訂製實現，再利用 Provider 提供全局網頁的明暗調換。時間較短則會直接在全局中多加暗色模式的樣式，採用 mix-blend-mode: difference 快速實現。
